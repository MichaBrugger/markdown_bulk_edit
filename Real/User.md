# User

A user is someone that uses a commodity or a service. This includes both people that pay for a [[Product|product]] or a [[Service|service]], and people that do not play (in [[Business Model, Two-Sided-Market|two-sided-market]] and [[Business Model, Freemium|freemium]] model).[^1]
